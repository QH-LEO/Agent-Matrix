# How To Query This Wiki

## Answer

查询本 Wiki 时，先读 `SCHEMA.md`、`index.md`、`log.md` 最近记录，然后从 `index.md` 找到相关稳定页面。只有当页面缺失、事实需要核验或出现冲突时，再查看 `sources/source-catalog.md`、`raw/` 或代码。

## Steps

1. Read `SCHEMA.md`.
2. Read `index.md`.
3. Read recent `log.md`.
4. Follow stable pages and `related:` links.
5. Cite stable pages and source ids.
6. If the answer is reusable, propose updating `queries/`.
